GitHub Actions 是獨立的，不受 Render 睡眠影響。

---

是**各專案分別計算**。

每個 Supabase 專案有自己獨立的休眠計時器，互不影響。

所以如果你有兩個 Supabase 專案：
- 專案 A 每天有人用 → 不會休眠
- 專案 B 超過 7 天沒有請求 → 會休眠

我們的 `hermes-cloud` 專案只要你有在用 Bot，就不會休眠。你之前那個舊專案是獨立的，需要另外保活。
